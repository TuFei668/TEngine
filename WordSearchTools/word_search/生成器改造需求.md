# Word Search Generator 改造需求文档

> 版本：1.0  
> 日期：2026-04-21  
> 作用：改造现有生成器，支持 Excel 驱动的关卡批量配置

---

## 一、功能目标

将生成器从「手动输入单词 → 生成」模式扩展为「Excel 驱动的关卡配置 → 生成 → 加密二进制存储」模式，同时**完整保留**原有手动模式。

核心流程：

```
Excel 文件 (Assets/Excel/*.xlsx)
    ↓ 扫描
Dropdown1: 大类型 (stage)   ← 来自 Excel 内 #大类型名 标签
    ↓ 级联
Dropdown2: 主题 (theme_en)  ← 来自 #大类型名 行的 B 列英文
    ↓ 级联
Dropdown3: 关卡 id          ← 来自 #大类型名 行的 D 列数字
    ↓
读取该关卡的 Words / HiddenWords / BonusWords
    ↓
判断 {项目父目录}/level_config/{pack_id}_{theme_en}/{pack_id}_{theme_en}_{level_id}.bytes 是否存在
    ├── 存在 → 解密 + 反序列化 → 展示已有网格
    └── 不存在 → 填词到 wordInputField + 自动生成一次
    ↓
用户可以反复点「Generate」重新生成，「Save」保存为加密二进制
```

---

## 二、Excel 数据结构

### 文件位置
`Assets/Excel/*.xlsx`（如 `Primary关卡数据.xlsx`、`Junior关卡数据.xlsx`）

### 表格布局（分块标签式，每关 4 行）

| 行 | A 列         | B 列起内容                                     |
| -- | ------------ | ---------------------------------------------- |
| 1  | `#大类型名`（如 `#Primary`） | `theme_en` / `theme_zh` / `level_id`（如 fruit / 水果 / 1） |
| 2  | `Words`      | 正确答案词（参与生成，彩色显示）               |
| 3  | `HiddenWords`| 隐藏词（参与生成，**深灰色**显示）             |
| 4  | `BonusWords` | Bonus 词（参与生成，**灰色**显示）             |

- **大类型 stage** 来自 `#` 开头行，去掉 `#` 即为原始字段名（如 `Primary` / `Junior`）
- **pack_id** = `stage.ToLower()`，无需映射表：`Primary` → `primary`，`Junior` → `junior`
- 颜色分组仅为视觉辅助，数据解析只认 `#大类型名` 标签切块

### 读取方案
**NPOI**（用户手动下载 DLL 放到 `Assets/Plugins/NPOI/`）

## 三、UI 改造

### 场景 / 预设
- 新场景：`GenScene`
- 新预设体：`GenWordSearchCanvas`
- **复用 `MainWindow.cs`**（不新建类），新增字段即可

### 新增 UI 控件（3 个级联 Dropdown + 1 个刷新按钮）
| 字段名             | 类型     | 约定路径（AutoBind 用）              |
| ------------------ | -------- | ------------------------------------ |
| `stageDropdown`    | Dropdown | `ExcelPanel/StageDropdown`           |
| `themeDropdown`    | Dropdown | `ExcelPanel/ThemeDropdown`           |
| `levelIdDropdown`  | Dropdown | `ExcelPanel/LevelIdDropdown`         |
| `refreshExcelButton` | Button | `ExcelPanel/RefreshExcelButton`（可选） |

### 原有控件全部保留
- 难度 Dropdown、useHard Toggle、SizeFactor Slider、交叉偏好 Toggle、网格尺寸 Dropdown
- wordInputField（改为**只读预览**，显示 Excel 读入的词；原手动输入功能保留但优先级低于 Excel）
- Generate / Cancel / SaveJson / SaveTxt / ViewAnswers 按钮

### 行为流程

**启动时：**
1. 扫描 `Assets/Excel/*.xlsx`，解析所有文件
2. 聚合出所有 `stage` 列表 → 填充 `stageDropdown`
3. 恢复 `PlayerPrefs` 中保存的上次三级选择索引
4. 触发 Dropdown 联动逻辑（加载或自动生成）

**用户切换 Dropdown 时：**
1. 立即保存到 `PlayerPrefs`（切换即记录）
2. 级联刷新下级 Dropdown
3. 三级都选定后：
   - 读取 Excel 对应关卡的 Words/HiddenWords/BonusWords
   - **合并去重**（去空格、去标点、同关卡内跨类别去重）
   - 把三类词分色填到 `wordInputField`（仅预览）
   - 检查 `{项目父目录}/level_config/{pack_id}_{theme_en}/{pack_id}_{theme_en}_{level_id}.bytes`
     - 存在 → 解密加载，展示已保存网格，**不触发生成**
     - 不存在 → 自动触发一次生成

**用户点 Generate：**
- 始终用**当前 Dropdown 对应的 Excel 数据**重新生成（不使用 wordInputField 文本）
- 覆盖当前内存中的谜题数据

**用户点 Save：**
- 把当前 `WordSearchData` 序列化为 JSON → 加密 → 写入 `.bytes`
- 覆盖同名文件，日志提示

## 四、生成逻辑

### 词合并
三类词合并成一个列表传给 `Generator.GenerateWordSearch()`：

```csharp
List<string> allWords = Dedup(words).Concat(Dedup(hidden)).Concat(Dedup(bonus)).ToList();
```

### 去重规则
1. 去首尾空格
2. 去除内部空格和标点符号（`[^A-Za-z]`）
3. 统一转大写
4. 同关卡内跨三类去重，优先级：**Words > BonusWords > HiddenWords**
5. 空字符串过滤

### 生成后回填
生成完成后，按类别拆分结果：
- `Words` 对应的词 → `wordPositions`（保持 `ColorManager` 彩色）
- `BonusWords` 对应的词 → `bonusWords`，颜色统一为 **灰色** `rgba(0.5, 0.5, 0.5, 0.6)`
- `HiddenWords` 对应的词 → `hiddenWords`，颜色统一为 **深灰色** `rgba(0.25, 0.25, 0.25, 0.6)`

## 六、文件存储

### 目录与命名
- **目录**：`{项目父目录}/level_config/{pack_id}_{theme_en}/`
- **文件名**：`{pack_id}_{theme_en}_{level_id}.bytes`
- **示例**：`level_config/primary_fruit/primary_fruit_1.bytes`

`{项目父目录}` = `Path.GetFullPath(Path.Combine(Application.dataPath, "..", ".."))`，与现有 `ProjectSiblingRoot` 一致。

### 存储流程（C 方案：真二进制）
```
WordSearchData
    ↓ JsonUtility.ToJson (明文 json 字符串)
    ↓ common_json_encrypt_tool.encrypt_json_string  (加密为字符串)
    ↓ Encoding.UTF8.GetBytes  (转原始字节)
    ↓ File.WriteAllBytes (*.bytes)
```

读取反向：`ReadAllBytes → UTF8.GetString → decrypt → FromJson`。

> 需要在 `common_json_encrypt_tool` 提供解密方法（若无需新增）。

### 保留 / 删除
- **删除**：原 `level_config_txt` 下的明文 json/txt（不再输出）
- **删除**：原 `level_config/{bookId}/configs/*.json` 加密字符串文件
- **保留**：原有的 `SavePuzzleAsJson` / `SavePuzzleAsTxt`（手动模式调试用，放到 `Application.persistentDataPath`，不碰项目目录）
- **新增**：`FileManager.SavePuzzleAsEncryptedBytes(data, packId, themeEn, levelId)`
- **新增**：`FileManager.LoadPuzzleFromEncryptedBytes(packId, themeEn, levelId)`
- **新增**：`FileManager.EncryptedBytesExists(packId, themeEn, levelId)`

### 覆盖策略
用户点 Save 时直接覆盖同路径文件，Debug.Log 提示。

## 七、记忆用户选择

使用 `PlayerPrefs`，Key 约定：
- `WSG_Last_StageIndex` (int)
- `WSG_Last_ThemeIndex` (int)
- `WSG_Last_LevelIdIndex` (int)
- `WSG_Last_ExcelSignature` (string，Excel 文件名+主题 hash，防止 Excel 改动后索引错位)

切换任何一级 Dropdown 时立即写入。启动时读取并恢复（若 Signature 不匹配则清空并用默认 0）。
