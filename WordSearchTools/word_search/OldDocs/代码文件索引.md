## 🔍 按功能分类

### 核心算法 (必读 ⭐⭐⭐⭐⭐)

#### 1. Generator.cs ⭐⭐⭐⭐⭐
**路径**: `Assets/WordSearchGenerator/Scripts/Core/Generator.cs`  
**行数**: ~550行  
**作用**: 核心生成器，实现回溯算法

**关键方法**:
- `GenerateWordSearch()` - 主生成方法
- `GetPuzzleDimension()` - 计算拼图尺寸
- `CanPlace()` - 检查能否放置
- `RenderPuzzle()` - 渲染拼图

**何时使用**: 
```csharp
Generator gen = new Generator();
WordSearchData puzzle = gen.GenerateWordSearch(words);
```

---

#### 2. Position.cs ⭐⭐⭐⭐
**路径**: `Assets/WordSearchGenerator/Scripts/Core/Position.cs`  
**行数**: ~150行  
**作用**: 表示单词位置

**关键方法**:
- `BoundsCheck()` - 边界检查
- `GetIndices()` - 获取坐标索引

**何时使用**: 生成器内部使用，通常不需要直接调用

---

#### 3. WordSearchData.cs ⭐⭐⭐⭐
**路径**: `Assets/WordSearchGenerator/Scripts/Core/WordSearchData.cs`  
**行数**: ~250行  
**作用**: 谜题数据模型

**关键方法**:
- `GridToString()` - 序列化
- `StringToGrid()` - 反序列化

**何时使用**: 
```csharp

// puzzle包含所有谜题数据
```

---

#### 4. Constants.cs ⭐⭐⭐
**路径**: `Assets/WordSearchGenerator/Scripts/Utils/Constants.cs`  
**行数**: ~150行  
**作用**: 定义所有常量

**关键常量**:
- `ALL_DIRECTIONS` - 8方向数组
- `EASY_DIRECTIONS` - 4方向数组
- `SIZE_FACTOR_DEFAULT` - 默认尺寸因子

**何时使用**:
```csharp
var dirs = Constants.ALL_DIRECTIONS;
```

### 数据管理 (重要 ⭐⭐⭐⭐)

#### 5. FileManager.cs ⭐⭐⭐⭐⭐

**行数**: ~300行
