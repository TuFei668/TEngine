### 第三步：查看生成的文件

#### Windows
1. 按 `Win + R`
2. 输入：`%USERPROFILE%\AppData\LocalLow`
3. 找到你的公司/游戏文件夹
4. 进入 `SavedPuzzles` 文件夹

#### Mac
1. 打开Finder
2. 按 `Cmd + Shift + G`
3. 输入：`~/Library/Application Support`
4. 找到你的公司/游戏文件夹
5. 进入 `SavedPuzzles` 文件夹

#### 或者直接在Console中查看路径
运行测试后，Console会显示：
```
保存目录: /path/to/SavedPuzzles
```

直接打开这个路径。

## 📝 代码使用示例

### 示例1：最简单的用法

创建一个新脚本 `SimpleTest.cs`:

```csharp
using UnityEngine;
using System.Collections.Generic;
using WordSearchGenerator;

public class SimpleTest : MonoBehaviour
{
    void Start()
    {
        // 1. 创建单词列表
        List<string> words = new List<string> { "HELLO", "WORLD" };
        
        // 2. 生成谜题
        Generator gen = new Generator();
        WordSearchData puzzle = gen.GenerateWordSearch(words);
        
        // 3. 输出结果
        Debug.Log(puzzle.puzzleText);
    }
}
```

### 示例2：完整配置

```csharp
using UnityEngine;
using System.Collections.Generic;
using WordSearchGenerator;

public class AdvancedTest : MonoBehaviour
{
    void Start()
    {
        // 准备单词
        List<string> words = new List<string> 
        { 
            "UNITY", "CSHARP", "GAME", "CODE", "DEBUG" 
        };
        
        // 创建生成器（带进度回调）
        Generator gen = new Generator(() =>
        {
            Debug.Log("正在生成...");
        });
        
        // 生成谜题（完整配置）
        WordSearchData puzzle = gen.GenerateWordSearch(
            words,                              // 单词列表
            Constants.ALL_DIRECTIONS,           // 8方向（困难）
            4,                                  // 尺寸因子
            Constants.INTERSECT_BIAS_PREFER     // 偏好交叉
        );
        
        // 输出信息
        Debug.Log($"拼图尺寸: {puzzle.dimension}x{puzzle.dimension}");
        Debug.Log($"单词数量: {puzzle.words.Count}");
        Debug.Log($"\n拼图:\n{puzzle.puzzleText}");
        Debug.Log($"\n答案:\n{puzzle.answerKeyText}");
        
        // 保存文件
        string jsonPath = FileManager.SavePuzzleAsJson(puzzle);
        string txtPath = FileManager.SavePuzzleAsTxt(puzzle);
        
        Debug.Log($"\n文件已保存:\nJSON: {jsonPath}\nTXT: {txtPath}");
    }
}
```

### 示例3：加载已保存的谜题

```csharp
using UnityEngine;
using WordSearchGenerator;

public class LoadTest : MonoBehaviour
{
    void Start()
    {
        // 获取最近的谜题
        string fileName = FileManager.GetMostRecentPuzzle();
        
        if (!string.IsNullOrEmpty(fileName))
        {
            // 加载谜题
            WordSearchData puzzle = FileManager.LoadPuzzleFromJson(fileName);
            
            if (puzzle != null)
            {
                Debug.Log($"加载成功: {fileName}");
                Debug.Log($"创建时间: {puzzle.createTime}");
                Debug.Log($"单词数量: {puzzle.words.Count}");
                Debug.Log($"\n拼图:\n{puzzle.puzzleText}");
            }
        }
        else
        {
            Debug.Log("没有找到已保存的谜题");
        }
    }
}
```

## ⚙️ 配置选项说明

### 方向模式

#### 简单模式（4方向）
```csharp
Constants.EASY_DIRECTIONS
```
- ✅ 右上 ↗
- ✅ 右 →
- ✅ 右下 ↘
- ✅ 下 ↓

**适合**: 儿童、初学者

#### 困难模式（8方向）
```csharp
Constants.ALL_DIRECTIONS
```
- ✅ 包含简单模式的4个
- ✅ 上 ↑
- ✅ 左 ←
- ✅ 左上 ↖
- ✅ 左下 ↙

**适合**: 成人、高级玩家

### 尺寸因子 (Size Factor)

```csharp
public static int GetPuzzleDimension(List<string> words, int sizeFactor)
{
    // 总字符数 = 单词字符数 × sizeFactor
    // 拼图边长 = √(总字符数)
}
```

**示例**:
- 单词: "HELLO", "WORLD" (5+5=10个字母)
- sizeFactor = 4
- 总字符数 = 10 × 4 = 40
- 拼图边长 ≈ √40 ≈ 6.3 → 7

**推荐值**:
- `1-2`: 非常紧凑（困难）
- `3-5`: 适中（推荐）
- `6-10`: 宽松（简单）

### 交叉偏好 (Intersect Bias)

```csharp
Constants.INTERSECT_BIAS_AVOID   // -1: 避免交叉
Constants.INTERSECT_BIAS_RANDOM  //  0: 随机（默认）
Constants.INTERSECT_BIAS_PREFER  //  1: 偏好交叉
```

**效果**:
- **Avoid**: 单词尽量不重叠
- **Random**: 随机放置
- **Prefer**: 单词尽量重叠（生成更快，拼图更小）

## 🐛 常见问题

### Q1: Console没有输出？
**A**: 确保：
1. `Run Test On Start` 已勾选
2. 已点击Play按钮
3. Console窗口已打开（Window → General → Console）
4. Console没有被清除（Clear按钮）

### Q2: 找不到保存的文件？
**A**: 查看Console输出的完整路径，或使用：
```csharp
Debug.Log(FileManager.GetSaveDirectoryPath());
```

### Q3: 生成速度慢？
**A**: 
- 减少单词数量（< 10个）
- 增大尺寸因子（4 → 6）
- 使用简单模式（4方向）

### Q4: 如何重新生成？
**A**: 
- 方法1: 再次点击Play
- 方法2: 右键组件 → Quick Test
- 方法3: 代码中调用 `QuickTest()`
